# Hello Word – Projeto Inicial
## Descrição

Este projeto foi criado pelos desenvolvedores como parte da configuração inicial do ambiente.
Ele imprime a mensagem "Hello Word!", servindo como base para testar setup, versionamento e fluxo de trabalho no GitHub.

## Tecnologias Utilizadas

Python

GitHub

## Como Rodar o Projeto

Certifique-se de ter o Python instalado na sua máquina.

### Clone o repositório:
```bash
git clone https://github.com/GiuseppCalderaro/Netara.git
```
### Acesse a pasta do projeto:
```bash
cd seu-repositorio
```
### Execute o arquivo principal:
```bash
python main.py
```
## Estrutura
```bash
/projeto
 ├── main.py     # Arquivo principal que imprime "Hello Word"
 └── README.md   # Documentação do projeto
```
## Funcionalidade

Exibe a mensagem "Hello Word" no console.

Serve como base inicial para futuros testes, validações e integrações.
